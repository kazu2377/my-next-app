ENTRY

ユニークな発想が
銀行を変える

1895年の創業から、新しいこと、違うことへ挑戦し、
「あってよかった、出会えてよかった。」を創造してきたスルガ銀行。

私たちはお客さまの想いを聴き、課題に真摯に向き合い、必要とされる価値を提供するために、
当事者意識をもって取り組みむことができる人を求めています。

▶ イベント情報: ＜2026年度新卒採用 エントリー受付 / 自社説明会実施中＞

応募希望の方は 「2026年度新卒採用」よりエントリーをお願いします。

企業理念

あってよかった、
出会えてよかった、
と思われる存在でありたい。

どんな時代が変わろうとも、
どんなに時が経とうとも、
お客さまの視点に立ったサービスを追求するとともに、
お客さまの人生に寄り添い、
必要とされる価値と、豊かな暮らしを提供します。
